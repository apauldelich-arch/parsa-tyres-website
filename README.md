# Parsa Tyres Services website

High-performance, agency-level website for Parsa Tyres Services Ltd, a tyre service business offering a 24/7 mobile tyre call-out service in Perivale and West London.

## Features

- **High-Conversion Layout**: Strategically designed to convert emergency roadside searchers into phone calls.
- **Mobile First UX**: Features a sticky mobile CTA button to ensure fast dialing on mobile platforms.
- **Performance-Oriented**: Built using pure HTML5 and Vanilla CSS for minimal load times and maximum responsiveness without relying on heavy frontend frameworks.
- **Premium Automotive Theme**: Tailored color palette with Performance Red `#E30613`, Black `#121212`, and Carbon Dark `#1C1C1C`.

## Project Structure

- `index.html`: The core single-page layout.
- `styles.css`: Custom design system, typography, animations, responsive grids, and variables.
- `assets/`: Contains premium AI-generated imagery for the Hero banner and Mobile Service sections.

## Technologies Used
- HTML5
- CSS3
- FontAwesome Icons (loaded via CDN)
- Google Fonts (Montserrat, Oswald, Inter)

## Deployment Options

This is a static website and can be deployed rapidly on free hosting platforms such as:

### 1. GitHub Pages
1. Push this repository to GitHub.
2. Go to **Settings** > **Pages**.
3. Set the source under 'Build and deployment' to `main` branch, `/root` folder.
4. Save, and your site will be live within minutes.

### 2. Netlify / Vercel
Simply drag and drop the main project folder into the Netlify/Vercel dashboard, or link your GitHub repository.
